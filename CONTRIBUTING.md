Thank you for considering contributing!

This project is primarily maintained by the ML Engineering Team. To keep
contributions high quality and safe for production, please follow these
guidelines:

1. Fork the repository and open a pull request against `main`.
2. Ensure your branch passes CI: linting and tests.
3. Keep changes focused and small; prefer multiple small PRs over one large PR.
4. Write tests for new behavior and update docs where appropriate.
5. If your change affects performance characteristics, include a benchmark or
   notes about the impact.

If this repository is internal-only, contact the ML Engineering Team for
contributor access.
